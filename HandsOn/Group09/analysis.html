Analyses of the selected data

At the csv file, we can see 30 different types of values

  1. "PK":

    Number that identifies the parking

  2. "NOMBRE"

    Name that specifies type of parking and the location of it

  3. "DESCRIPCION-ENTIDAD"

    Extra description of the parking

  4. "HORARIO"

    Schedule of the parking

  5. "EQUIPAMIENTO"

    All the fields are empty

  6. "TRANSPORTE"

    All the different near public transport media

  7. "DESCRIPCION"

    Number and type of parking places

  8. "ACCESIBILIDAD"

    Tt can be 0 or 1 depending on if it allows people with reduced mobility to use it

  9. "CONTENT-URL"

    The URL in madrid.es domain

  10. "NOMBRE-VIA"

    The name of the via where the parking is in

  11. "CLASE-VIAL"

    Determinates if the "NOMBRE-VIA" field is a street, an avenue or a square

  12. "TIPO-NUM"

    Type of number the "NUM" uses

  13. "NUM"

    Number of the street of the parking entrance

  14. "PLANTA" "PUERTA" or "ESCALERAS"

    all empty

  15. "ORIENTACION"

    Where the parking is looking at

  16. "LOCALIDAD"

    Town where the parking is

  17. "PROVINCIA"

    Province where the parking is

  18. "CODIGO-POSTAL"

    Postal code of the locality or disctrict where the parking is

  19. "BARRIO"

    Neighborhood where the parking is

  20. "DISTRITO"

    District of the parking

  21. "COORDENADA-X"

    Parking position at the X coordinate

  22. "COORDENADA-Y"

    Parking position at the Y coordinate

  23. "LATITUD"

    Latitude of the parking

  24. "LONGITUD"

    Longitude of the parking

  25. "TELEFONO"

    Telephone number of the parking

  26. "FAX"

    FAX number of the parking

  27. "EMAIL"

    Email address of the parking

  28. "TIPO"

    Describes if the parking is public or for residents

Appropriated License

The license we think might be the most appropriate is the Open Data Commons Attribution License. This is because the open data of Madrid requires the attribution of the source to the city of Madrid.

Naming strategy

Since there are some fields always empty we will ignore them.

The main domain would be https://publicparkingmad.com/

  • https://publicparkingmad.com/[PK]
    PK is the resource that a parking represents
  • https://publicparkingmad.com/[PK]/[Place]
    Place represents where the parking is located
  • https://publicparkingmad.com/[district]
    District shows all the parkings in a specific district
