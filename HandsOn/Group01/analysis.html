The analyses performed over the data source

Los datos que hemos analizado han sido obtenidos de 2 CSV.
En primer lugar tenemos los datos de Centros Educativos de Madrid:

  • Identificador: valor String de 8 cifras, empieza por 28
  • Nombre: valor de tipo String
  • Tipo: valor String, los distintos tipos que tenemos en este apartado son centros privados, públicos o concertados que pueden tener formación Infantil, Primaria, Secundaria o Bachillerato.
  • Titularidad: valor String, pueden ser [PRIVADO, PRIVADO CONCERTADO, PÚBLICO, PÚBLICO-TITULARIDAD PRIVADA]
  • Municipio: valor String de [001-183]
  • Coordenada_X: valor String de 6 cifras
  • Coordenada_Y: valor String de 7 cifras
  • Página Web: valor anyURI. Ejemplo: "http://www.w3.org/TR/xmlschema-0"
  • Dirección:
    • Nombre: valor String
    • Número: valor String. En el caso de que no tenga número toma el valor de "s/n"
    • Código Postal: valor String que empieza por 28
    • Tipo de vía: valor String

Por último, tenemos los datos de los Los Residentes de la Comunidad de Madrid por Edades:

  • Identificador: valor String de [001-183]. Es el del municipio al que pertenecen
  • Nombre: valor String. Es el del municipio al que pertenecen
  • Sexo: valor String, puede ser [Mujer, Hombre]
  • Edad mínima: valor numérico entero siendo el mínimo valor el 0
  • Edad máxima: valor numérico entero siendo el máximo valor el 199
  • Población: valor numérico entero, de [0-98949]
